# FCPO Trading Bot

Algorithmic trading bot for FCPO Futures on Bursa Malaysia Derivatives.

## Overview
- Broker: CGS International (CGS CIMB)
- Market: FCPO (Crude Palm Oil Futures) - Bursa Malaysia
- Strategy: TradingView signal-based execution

## Planned Features
- TradingView webhook integration
- Automated order execution via broker API
- Risk management (stop-loss, position sizing)
- Trade logging and P&L tracking
